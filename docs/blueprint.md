# **App Name**: TaxWise UK

## Core Features:

- Tax Calculation: Calculate income tax, NICs, and take-home pay based on various inputs like salary, bonus, benefits, and tax code.
- Scenario Planning: Allow users to adjust input values and see changes to the estimated output.
- Regional Support: Handle tax rules and rates specific to UK regions like Scotland and Wales.
- Tax Saving Tips Tool: AI-powered tool offering suggestions for reducing taxable income, such as pension contributions or salary sacrifice. Generates personalized financial advice based on user income details.
- Data Persistence: Store scenarios based on User profile information.
- Threshold Alerts: Alert users when their income approaches critical thresholds like £50,000 or £100,000, offering insights into potential tax implications.
- Backdated Pay Calculations: Accurately recalculate tax and NICs for previous periods due to backdated pay increases or corrections.

## Style Guidelines:

- Primary color: Deep blue (#003049) to convey trust and financial expertise.
- Background color: Light grey (#F0F0F0), providing a clean and neutral backdrop.
- Accent color: Bright orange (#D62828) for highlighting important actions and warnings.
- Body font: 'Inter', a sans-serif font, for a clean and neutral feel. Headline font: 'Space Grotesk', a sans-serif font, for short titles and emphasis.
- Use simple, clean icons to represent various income and deduction categories.
- Employ a clear, structured layout with intuitive input forms and easily digestible results displays.
- Subtle animations for transitions and feedback to enhance user interaction.
